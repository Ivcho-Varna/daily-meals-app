Курсов проект по "Разработка на уеб приложения"

Студент: Иво Георгиев

## Изисквания
- min node v21.6.1
- min npm 10.2.4

## Инсталация на npm пакети

```sh
npm install
```

### Компилиране на проекта за development

```sh
npm run dev
```

### Компилиране на проекта за production

```sh
npm run build
```

### Стартиране на node сървър за преглед

```sh
npm run preview
```
